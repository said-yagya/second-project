# 🌤️ Приложение Погоды для Санкт-Петербурга

Приложение для получения актуальных данных о погоде в Санкт-Петербурге с сайта AccuWeather.

## 📊 Получаемые данные

- 🌡️ **Температура** (текущая и ощущается как)
- 💨 **Ветер** (скорость в км/ч и направление)
- 🌡️ **Атмосферное давление** (в мм рт. ст.)
- 💧 **Влажность воздуха** (в процентах)
- ☁️ **Состояние погоды** (ясно, облачно, дождь и т.д.)

## 🚀 Быстрый старт

### Веб-приложение

1. Откройте файл `index.html` в браузере
2. Приложение автоматически загрузит данные о погоде
3. Используйте кнопку "Обновить" для получения свежих данных

### Python-скрипты

1. Установите зависимости:
```bash
sudo apt install -y python3-requests python3-bs4 python3-lxml
```

2. Запустите один из скриптов:
```bash
# Веб-скрапинг AccuWeather
python3 weather_scraper.py

# Или через OpenWeatherMap API
python3 weather_api.py

# Или используйте удобный скрипт запуска
./run.sh
```

## 🔧 Настройка

### Для реальных данных в веб-приложении

1. Получите API ключ на [OpenWeatherMap](https://openweathermap.org/api)
2. В файле `script.js` замените:
```javascript
API_KEY: 'demo_key'
```
на:
```javascript
API_KEY: 'ваш_api_ключ'
```

### Для Python-скрипта

Python-скрипт работает напрямую с AccuWeather без API ключей, используя веб-скрапинг.

## 🔄 Автоматизация

### Cron для Linux/macOS
Добавьте в crontab для автоматического получения данных каждый час:
```bash
0 * * * * /usr/bin/python3 /path/to/weather_scraper.py
```

### Task Scheduler для Windows
Создайте задачу в планировщике заданий для регулярного запуска скрипта.

## ⚠️ Важные замечания

1. **CORS-ограничения**: Веб-версия может иметь ограничения при запросах к внешним API
2. **Rate Limiting**: Не злоупотребляйте частотой запросов к AccuWeather
3. **Изменения сайта**: Структура AccuWeather может изменяться, что потребует обновления парсера
4. **API ключи**: Для production использования получите собственные API ключи

## 🚨 Устранение неполадок

### Веб-приложение не загружает данные
- Проверьте консоль браузера на ошибки
- Убедитесь, что API ключ корректный (если используете реальный)
- Проверьте подключение к интернету

### Python-скрипт не работает
- Установите все зависимости: `pip install -r requirements.txt`
- Проверьте подключение к интернету
- Убедитесь, что AccuWeather доступен

### Данные не парсятся
- AccuWeather мог изменить структуру сайта
- Проверьте логи ошибок
- В таком случае будут показаны демо-данные
